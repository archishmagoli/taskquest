# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. **User** - A person who uses TaskQuest to manage their tasks, earn points, and customize their profile with unlocked avatars.

## Draft User Stories

1. As a user, I want to create a new task with a title, description, due date, and category, so that I can keep track of everything I need to do.
2. As a user, I want to view all of my tasks in a list, so that I can see what needs to be done at a glance.
3. As a user, I want to mark a task as complete, so that I can track my progress and feel a sense of accomplishment.
4. As a user, I want to automatically earn points when I complete a task, so that I am rewarded for my productivity without any extra steps.
5. As a user, I want to edit an existing task's details, so that I can update it if something changes.
6. As a user, I want to delete a task, so that I can remove tasks I no longer need.
7. As a user, I want to browse the Avatar Shop and see all available avatars along with their point costs, so that I know what I can unlock and what to work toward.
8. As a user, I want to unlock an avatar by spending my earned points, so that I can personalize my profile as a reward for completing tasks.
9. As a user, I want to equip any avatar I have already unlocked, so that it displays on my profile.
10. As a user, I want to view my profile page showing my current avatar, total points, and completion stats, so that I can see an overview of my progress.
11. As a user, I want to filter my tasks by category, so that I can focus on a specific area of my life like school or work.
12. As a user, I want to sort my tasks by due date, so that I can prioritize the most urgent items first.
13. As a user, I want to reset the database to its default state, so that I can start fresh if needed.
